# TMDB_movie_database

This project is a part of the Data Analyst Nanodegree. This project is to analyse the TMDB movie dataset which contains data of 10000 movies. Here we have analyse the depending factors of a successful movie and all other relationships between other varying factors.

## Getting Started:
For this projects one requires the following python libraries -
> * numpy
> * pandas
> * matplotlib
> * seaborn
> * plotly
> * wordcloud

One can install these libraries using `pip` or `conda` command like `pip install package_name`
